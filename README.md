# AliasLab — 采样混叠与重建智能仿真系统

面向数字信号处理教学的采样-混叠-重建全过程智能分析平台。

## 功能简介

- **信号生成**: 支持单频/双频正弦波、方波、含噪信号
- **采样仿真**: 可调采样率，动态展示过采样、临界采样、欠采样
- **混叠可视化**: 自动计算混叠频率，在频谱图上标注频率折叠
- **FFT 频谱分析**: 原始信号 vs 采样后频谱对比
- **sinc 重建**: 基于 sinc 插值的信号重建，含误差分析
- **智能分析 Agent**: 基于规则推理的 DSP 实验助手，自动诊断采样质量并给出优化建议

## DSP 理论说明

### 奈奎斯特采样定理

要从采样后的离散信号无失真地重建原始连续信号，采样率必须满足：

```
fs ≥ 2·fmax
```

其中 fmax 为信号的最高频率分量。2·fmax 称为**奈奎斯特采样率**，fs/2 称为**奈奎斯特频率**。

### 混叠 (Aliasing)

当 fs < 2·fmax 时，高于 fs/2 的频率分量会被"折叠"到低频区：

```
f_alias = |f_signal - k·fs|,  取 k 使得 f_alias ∈ [0, fs/2]
```

混叠导致高频信号在采样后表现为虚假的低频信号，无法通过任何后处理消除。

### sinc 插值重建

满足奈奎斯特条件时，可通过 sinc 插值精确重建原始信号：

```
x(t) = Σ x[n] · sinc((t - n·Ts) / Ts)
```

其中 sinc(x) = sin(πx) / (πx)，Ts = 1/fs。

### 采样状态分类

| 采样比 fs/fmax | 状态 | 说明 |
|:---:|------|------|
| < 1 | 严重欠采样 | 采样率低于信号频率，完全混叠 |
| 1~2 | 欠采样 | 不满足奈奎斯特定理，部分混叠 |
| ~2 | 临界采样 | 刚好满足定理，无余量 |
| 2~5 | 正常采样 | 满足定理，可正确重建 |
| > 5 | 过采样 | 远高于需求，重建质量极好 |

### 方波的带宽

方波由基频和奇次谐波组成：f, 3f, 5f, 7f, 9f, ...

虽然理论上方波带宽无限，但实际中主要能量集中在低次谐波。本系统考虑至 9 次谐波来计算等效带宽和奈奎斯特要求。

## 使用指南

1. 在左侧边栏选择信号类型并调节参数
2. 调节采样率 fs，观察采样点和频谱的变化
3. 尝试将 fs 降到 2·fmax 以下，观察混叠现象
4. 查看智能分析报告中的诊断结果和优化建议
5. 对比原始信号与 sinc 重建信号，理解采样定理的实际意义

### 推荐实验场景

- **正常采样**: 单频 50 Hz，fs = 200 Hz → 观察完美重建
- **临界采样**: 单频 50 Hz，fs = 100 Hz → 观察临界状态
- **混叠演示**: 双频 (40 + 150) Hz，fs = 100 Hz → 150 Hz 混叠至 50 Hz
- **方波采样**: 基频 10 Hz，fs = 60 Hz → 谐波混叠导致方波变形
- **噪声影响**: 含噪正弦波，对比不同噪声水平下的重建质量
